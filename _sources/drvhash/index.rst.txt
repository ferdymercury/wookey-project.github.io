The Hash driver
===============

.. highlight:: c


This library is an implementation of the STM32F4 Hash cryptographic IP driver.

It provide an abstraction of the Hash device interactions through high level
API in order to request hashing of any content, using the various hash methods
supported by the HASH coprocessor of the STM32F4 SoC.
